# Консольное приложение "Библиотека"

CRUD-приложение для управления каталогом библиотеки с возможностью добавления книг, их выдачи и возврата. Проект демонстрирует принципы объектно-ориентированного программирования и работу с пользовательскими исключениями.

## Функции
*   Просмотр всего каталога книг
*   Добавление новых книг с указанием названия, автора и количества копий
*   Выдача книги читателю (уменьшение количества доступных копий)
*   Возврат книги в библиотеку (увеличение количества доступных копий)
*   Обработка ошибок: книга не найдена, нет доступных копий, некорректный ввод

## Структура проекта
*   `Book` — класс, описывающий сущность книги (название, автор, количество доступных копий)
*   `Library` — основной класс, управляющий каталогом книг и содержащий логику выдачи/возврата
*   `ItemNotFoundException` — исключение, возникающее при попытке работы с несуществующей книгой
*   `NoAvailableCopiesException` — исключение, возникающее при попытке выдать книгу, которой нет в наличии
*   `Main` — класс с точкой входа, реализующий консольный пользовательский интерфейс
